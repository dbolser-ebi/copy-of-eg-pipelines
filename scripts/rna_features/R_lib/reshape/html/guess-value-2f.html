guess_value {reshape}R Documentation

Guess value

Description

Guess name of value column

Usage

guess_value(df)

Arguments

df

Data frame to guess value column from

Details

Strategy:

  1. Is value or (all) column present? If so, use that

  2. Otherwise, guess that last column is the value column

sort_df {reshape}R Documentation

Sort data frame

Description

Convenience method for sorting a data frame using the given variables.

Usage

sort_df(data, vars=names(data))

Arguments

data

data frame to sort

vars

variables to use for sorting

Details

Simple wrapper around order
